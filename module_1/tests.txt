Проверить удаление товаров из корзины

-- Удаление товара из корзины с несколькими товарами 2.1.1
Предусловия:
В корзину добавлено 2 товара, пользователь находится на странице корзины
Шаги:
- Шаг 1, нажать кнопку "удалить" на первом товаре из списка, ОР - сумма товаров в корзине уменьшится на цену удалённого товара, товар удалиться из списка товаров
-- Удаление всех товаров из корзины 2.1.2
Предусловия:
В корзину добавлен 1 товар, пользователь находится на странице корзины
Шаги:
- Шаг 1, нажать кнопку "удалить" на товаре из списка, ОР - сумма товаров в корзине станет равна 0, список товаров в корзине расформируется
